# Job Offer System - User Guide

This guide explains how to use the new Job Offer System within the IKAI HR Platform.

## 1. Creating an Offer

1.  Navigate to the **Offers** section from the main menu.
2.  Click the **+ Yeni Teklif** button.
3.  You can either choose to create from a template or start from scratch.
4.  Fill in the candidate details, position, salary, and other terms.
5.  Save the offer as a draft.

## 2. Approval Workflow

- Once an offer is created, it will be in **Taslak** (Draft) status.
- From the offer detail page, you can request approval. This will change the status to **Onay Bekliyor** (Pending Approval).
- A manager or admin must then approve the offer. Once approved, the status will change to **Onaylandı** (Approved).
- Only approved offers can be sent to candidates.

## 3. Sending an Offer

- On the detail page of an approved offer, click the **Gönder** (Send) button.
- The system will generate a PDF and email it to the candidate.
- The offer status will change to **Gönderildi** (Sent).

## 4. Bulk Sending

- From the main offers list, you can select multiple **approved** offers.
- A **Toplu Gönder** (Bulk Send) button will appear.
- Clicking this will queue all selected offers to be sent to their respective candidates.

## 5. Tracking and History

- **Negotiations**: On the offer detail page, you can view the negotiation history and add new messages.
- **Attachments**: You can upload or delete attachments related to the offer.
- **Revisions**: Click on the **Versiyon Geçmişi** (Version History) link to see a complete audit trail of all changes made to the offer.

## 6. Analytics

- Navigate to the **Offers -> Analytics** sub-section.
- Here you can view charts and statistics for:
  - Overall offer status distribution.
  - Acceptance rates.
  - Average response times.
  - Department-specific statistics.
